# Visual Testing Backend - Phase 1\n\nAI-Powered Visual Testing Platform Backend - Foundation Setup\n\n## Features Implemented (Phase 1)\n\n✅ **Core Infrastructure**\n- TypeScript project with Express\n- Zod configuration validation\n- Winston structured logging\n- Custom error handling\n- ESLint and Jest setup\n\n✅ **Database Layer**\n- MongoDB connection with Mongoose\n- Project and Baseline domain models\n- Repository pattern implementation\n- Data validation with Zod\n\n✅ **API Endpoints**\n- `GET /api/health` - Health check\n- `POST /api/projects` - Create project\n- `GET /api/projects` - List all projects\n- `GET /api/projects/:id` - Get project by ID\n- `PUT /api/projects/:id` - Update project\n- `DELETE /api/projects/:id` - Delete project\n- `POST /api/baselines` - Create baseline\n- `GET /api/projects/:projectId/baselines` - Get project baselines\n- `GET /api/baselines/:id` - Get baseline by ID\n- `DELETE /api/baselines/:id` - Delete baseline\n\n## Setup Instructions\n\n### Prerequisites\n- Node.js 18+\n- MongoDB (local or Atlas)\n- npm or yarn\n\n### Installation\n\n1. **Install dependencies**\n   ```bash\n   npm install\n   ```\n\n2. **Configure environment**\n   ```bash\n   cp .env .env.local\n   # Edit .env.local with your MongoDB URI\n   ```\n\n3. **Start MongoDB** (if running locally)\n   ```bash\n   mongod\n   ```\n\n4. **Run development server**\n   ```bash\n   npm run dev\n   ```\n\n5. **Run tests**\n   ```bash\n   npm test\n   ```\n\n### API Usage Examples\n\n**Create a Project**\n```bash\ncurl -X POST http://localhost:3000/api/projects \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"name\": \"My Website\",\n    \"baseUrl\": \"https://example.com\",\n    \"diffThreshold\": 95,\n    \"aiEnabled\": true\n  }'\n```\n\n**Create a Baseline**\n```bash\ncurl -X POST http://localhost:3000/api/baselines \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"projectId\": \"project-uuid\",\n    \"name\": \"homepage\",\n    \"image\": \"data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAYAAAAfFcSJAAAADUlEQVR42mP8/5+hHgAHggJ/PchI7wAAAABJRU5ErkJggg==\",\n    \"viewport\": { \"width\": 1920, \"height\": 1080 },\n    \"url\": \"https://example.com\"\n  }'\n```\n\n**Health Check**\n```bash\ncurl http://localhost:3000/api/health\n```\n\n## Project Structure\n\n```\nvisualTestBackend/\n├── core/                 #
